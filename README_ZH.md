# 🎄 你的专属豪华 3D 圣诞树

这是一个已经配置好的、可直接部署的“电子圣诞贺卡”项目。它包含了精美的 3D 圣诞树效果，并且已经设置为**加载你指定的本地照片**。

## 📸 怎么添加照片？

1. 打开本项目中的 `public/photos` 文件夹。
2. 将你想要展示的照片（建议为 JPG 格式）放入该文件夹。
3. **重要：** 请将照片依此重命名为 `1.jpg`, `2.jpg`, `3.jpg` ... 直到 `22.jpg`。
   - 系统默认支持 22 张位置。
   - 如果你的照片少于 22 张，也没关系，没有照片的位置会显示通用的占位符。
   - 建议尽量放满 15-22 张，效果最好。

## 🎵 怎么添加背景音乐？

1. 准备一首你喜欢的音乐，格式最好是 **MP3**。
2. 将文件重命名为 `music.mp3`。
3. 放入 `public` 文件夹中（和照片文件夹 `photos` 同级）。
4. **功能说明**：浏览器禁止网页自动播放声音。所以音乐**默认静音**，只有当用户**开启摄像头并检测到手势**时，或者**点击右下角的音乐按钮**时，才会开始播放。

## 🚀 怎么发布给别人看？

最简单的方法是使用 [Vercel](https://vercel.com/) 免费部署。

1. **上传代码到 GitHub**
   - 在 GitHub 上新建一个仓库（例如 `my-christmas-tree`）。
   - 将本项目代码推送到该仓库。

2. **在 Vercel 上导入**
   - 注册/登录 Vercel。
   - 点击 "Add New..." -> "Project"。
   - 选择你刚才创建的 GitHub 仓库。
   - 点击 "Deploy"。

3. **获取链接**
   - 等待约 1 分钟，部署完成后，Vercel 会给你一个链接（例如 `https://my-christmas-tree.vercel.app`）。
   - 把这个链接发给你的朋友即可！

## ✨ 功能与操作

- **双模式切换**：点击右下角按钮，可在“混沌散落(Chaos)”和“聚合成树(Formed)”之间切换。
- **手势控制**：如果你开启摄像头，可以用手势控制视角（张开手掌散开，握拳聚合）。
- **鼠标控制**：拖拽旋转，滚轮缩放。
